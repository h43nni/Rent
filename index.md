# Rent Service Specification\n\n## Introduction\nThe Rent service is designed to facilitate the management of rental agreements and transactions.\n\n## Features\n- **Tenant Management:** Manage tenant records including personal information and lease agreements.\n- **Property Listing:** List available properties for rent with details and photos.\n- **Payment Processing:** Handle payments securely with support for multiple payment methods.\n- **Reporting:** Generate reports on rental income, occupancy rates, and tenant statistics.\n\n## API Endpoints\n### 1. Get All Properties\n**GET** `/api/properties`\n- Returns a list of all properties available for rent.\n\n### 2. Get Property Details\n**GET** `/api/properties/{id}`\n- Returns detailed information about a specific property.\n\n### 3. Create Rental Agreement\n**POST** `/api/rental-agreements`\n- Creates a new rental agreement for a property.\n\n## Usage\n1. Authenticate using the API token provided during account creation.\n2. Use the endpoints to interact with the Rent service.\n\n## Conclusion\nThe Rent service aims to simplify the rental management process for landlords and tenants, ensuring a seamless experience.
